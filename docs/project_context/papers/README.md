# Related Papers

Drop related paper PDFs here.

Recommended filename pattern:

```text
FirstAuthor_Year_short-title.pdf
```

If you have citation metadata, add it here too:

- `references.bib` for BibTeX entries
- `paper_notes.md` for why each paper matters to the project

Codex can extract text, summarize each paper, and update `../manifest.md` after files are added.
